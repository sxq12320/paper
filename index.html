ℹ️ 关于 Research Hub

一个为计算机视觉研究者打造的论文管理工具

✨ 功能说明

  • 多级分类管理（检测 / 分割 / Transformer 等）
  • 画板式论文卡片，含作者、时间、摘要等属性
  • 实时搜索与筛选，支持星标收藏
  • 研究思考板块，记录选题灵感
  • JSON 导入/导出，轻松备份数据
  • 数据存储于浏览器 LocalStorage，刷新不丢失
💡 GitHub 部署提示：将 index.html 推送到仓库并开启 GitHub Pages，即可在线访问。
